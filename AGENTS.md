# AI Agent Instructions for Qubekit

Qubekit is a Node.js package/SDK that provides a unified, auto-generated API to
facilitate communication with SonarQube's web administration APIs.

## Essentials

- **Package Manager**: This project uses **Yarn** with Corepack enabled. Do not
  use npm.
- **Build / Test Commands**: Run `yarn build` and `yarn test` to verify types
  and functionality.
